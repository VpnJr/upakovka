# Upakovka09 — Инструкция

## Файлы
```
index.html      — Главная (каталог)
product.html    — Страница товара
checkout.html   — Оформление заказа
admin.html      — Панель администратора
css/style.css   — Стили
js/data.js      — Данные товаров (localStorage)
js/cart.js      — Корзина
js/catalog.js   — Каталог
js/admin.js     — Логика админки
```

## Публикация на GitHub Pages

1. Создайте репозиторий на github.com (Public)
2. Загрузите все файлы через "uploading an existing file"
3. Settings → Pages → Source: main → Save
4. Сайт: https://ВАШ_ЛОГИН.github.io/upakovka09/

## Смена пароля администратора

Откройте `js/admin.js`, первая строка:
```js
var ADMIN_PASSWORD = 'admin123';  // ← замените
```

## Панель администратора

Откройте: https://ВАШ_ЛОГИН.github.io/upakovka09/admin.html

Возможности:
- Просмотр и управление заказами (принять / выполнить / удалить)
- Фильтрация по статусу
- Добавление новых товаров
- Редактирование существующих товаров
- Удаление товаров
- Изменения сразу отражаются на сайте

## Как работают данные

Товары и заказы хранятся в localStorage браузера.
Это значит: покупатель и администратор должны открывать сайт
в ОДНОМ браузере на ОДНОМ компьютере.

Для реального магазина нужен бэкенд (Telegram-бот или база данных).

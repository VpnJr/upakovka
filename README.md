## Как опубликовать на GitHub Pages

### Шаг 1 — Создайте репозиторий
1. Откройте https://github.com и войдите в аккаунт
2. Нажмите зелёную кнопку **New** (новый репозиторий)
3. Название: `upakovka09` (или любое другое)
4. Тип: **Public** (это важно для бесплатного хостинга)
5. Нажмите **Create repository**

### Шаг 2 — Загрузите файлы
**Способ A — через браузер (проще):**
1. В созданном репозитории нажмите **uploading an existing file**
2. Перетащите ВСЕ файлы и папки (`index.html`, `checkout.html`, `admin.html`, папки `css/` и `js/`)
3. Нажмите **Commit changes**

**Способ B — через Git (если установлен):**
```bash
git init
git add .
git commit -m "Первая версия сайта"
git remote add origin https://github.com/ВАШ_ЛОГИН/upakovka09.git
git push -u origin main
```

### Шаг 3 — Включите GitHub Pages
1. В репозитории перейдите в **Settings** (шестерёнка)
2. Слева найдите раздел **Pages**
3. В поле **Source** выберите: `Deploy from a branch`
4. Branch: `main`, папка: `/ (root)`
5. Нажмите **Save**

### Шаг 4 — Готово!
Через 1–2 минуты сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/upakovka09/
```

## Важные настройки перед публикацией

### 1. Смените пароль администратора
Откройте файл `js/admin.js` и измените строку:
```javascript
const ADMIN_PASSWORD = 'upakovka2025';  // ← ЗАМЕНИТЕ НА СВОЙ ПАРОЛЬ
```

### 2. Как открыть панель администратора
Перейдите по адресу:
```
https://ВАШ_ЛОГИН.github.io/upakovka09/admin.html
```
Введите пароль из `admin.js`.

### 3. Как заказы попадают в панель
Покупатель оформляет заказ → заказ сохраняется в localStorage браузера.

⚠️ **Важно:** localStorage работает только в рамках ОДНОГО браузера.
Это значит: если покупатель оформил заказ на своём телефоне,
а вы открываете admin.html на своём компьютере — заказ не появится.

**Решение для реального магазина:** подключить Telegram-бот.
Напишите "Хочу Telegram-бот для заказов" — и получите готовый код.

## Как добавить новый товар

Откройте `js/products.js` и добавьте объект в массив `PRODUCTS`:
```javascript
{
  id: 9,                          // уникальный номер
  name: 'Название товара',
  meta: 'Описание / материал',
  price: 5.50,                    // цена за штуку
  oldPrice: 7.00,                 // старая цена (или null)
  cat: 'containers',              // containers / cups / bags / plates
  badge: 'new',                   // hit / new / eco / sale / null
  emoji: '📦'                     // эмодзи для иконки
},
```

## Как изменить контакты

Откройте `index.html` и найдите секцию `КОНТАКТЫ`:
```html
<div class="contact-item">📞 <a href="tel:+79380000000">+7 (938) 000-00-00</a></div>
<div class="contact-item">✉️ <a href="mailto:info@upakovka09.ru">info@upakovka09.ru</a></div>
```
Замените на ваши данные.
